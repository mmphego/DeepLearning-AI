Lesson 1: Introduction to Deep Learning.

What is a Neural Network?

The term Deep Learning refers to Neural Networks, and a Neural Network is an adaptive system that learns by using interconnected nodes or neurons in a layered structure that resembles a human brain. It learns from data input, such that it can be trained to recognize patterns, classify data and forecast future events. A great examples is the image below where by we are trying to predict housing prices based on the inputs:

  • Size
  • No. of bedrooms
  • Postal code / location and,
  • Wealth

With these input features we could be able to predict the values of y which is our price. The middle or hidden layer are called neural network and each of them takes their inputs from all 4 input features then computes and output the price.

Supervised Learning with Neural Networks

In Supervised Learning, you have an input x and you want a function mapping to output y. The example above we saw the housing price prediction app where you input some features of a home and try to output or estimate the price y.

The image below shows some examples that NN have been applied to very effectively.

It turns out that different types of NN are useful for diferent applications, for example:

  • Real Estate/ Online Ads → Standard NN
  • Photo Tagging → Convolutional NN (CNN)
  • Speech Recognition, Machine Translation → Recurrent NN (RNN)
  • Autonomous Driving → Complex/Hybrid CNN

Why is Deep Learning taking off?

  • Faster computation has increased the speed of training NN which improves feedback.
  • The increase of computational data has also increased.
  • Algorithmic innovations or improvement on old algorithms: The switching from Sigmoid to RELU activation function has made an Algorithm called Gradient Descent much faster.
